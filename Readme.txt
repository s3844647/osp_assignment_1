This simulation can be compiled using Make. You should add a condition after the command 'make' to run the simulation of your choice.

For the producer-consumer problem type 'make prodcon'.
For the sleeping barbers problem type 'make barbers'.
Typing just 'make' or 'make all' will lead to the producer-consumer problem running.
